A universal, high-fidelity, task-oriented robot simulation framework designed for reinforcement learning research in robotics.

## Abstract
This work presents UniRoboSim, a comprehensive simulation framework that enables high-fidelity robot simulation for various robotic tasks. The framework supports multiple robot platforms and provides a unified interface for reinforcement learning algorithms.

## Key Contributions
- Universal simulation framework supporting multiple robot platforms
- High-fidelity physics simulation for realistic robot behavior
- Task-oriented design for efficient RL training
